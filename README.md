# BookHub – Bookstore Management System

BookHub is a bookstore management system developed in Java using Object-Oriented Programming (OOP) concepts.
The system helps manage books, customers, orders, and inventory in an organized way.

##  Project Description

BookHub is designed to simulate a real bookstore system where users can browse books, manage orders, and handle inventory.
The project demonstrates key OOP principles such as modular design, classes, and system controllers.

##  Features

* User authentication (login / registration)
* Book inventory management
* Customer and supplier management
* Shopping cart functionality
* Order processing system
* Book review system
* Admin dashboard for managing the system

##  Technologies Used

* Java
* Object-Oriented Programming (OOP)
* JDBC
* Java GUI
